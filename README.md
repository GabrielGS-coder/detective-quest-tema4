# Detective Quest – Tema 4: Estrutura de Dados

Autor: Gabriel  
Linguagem: Python  

## Descrição
Este projeto foi desenvolvido como parte do Desafio Detective Quest – Tema 4, com foco em Estrutura de Dados.  
O programa implementa um sistema de investigação de suspeitos, utilizando dicionários em Python para armazenar e gerenciar informações.

## Funcionalidades
- Adicionar novos suspeitos (nome, idade, cidade)  
- Buscar suspeito pelo nome  
- Listar todos os suspeitos cadastrados  
- Remover suspeitos do sistema  

## Estrutura de Dados Utilizada
- Dicionário (`dict`): utilizado para armazenar os dados dos suspeitos, permitindo acesso rápido por nome.
